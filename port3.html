Overview



The method​ findNeedles takes two parameters, a string and a string array, compares them, and collects the matching elements to print each element, along with the count.

Method findNeedles


This method accepts the parameters​ haystack ​and​ needles.

public static void findNeedles(String haystack, String[]needles) {

When the function runs, it first checks for more than five elements in​ needleswith an if-else condition using the​ needles.length method​. ​ It generates an error message “Too many words!” if the check returns true.Otherwise, the else condition executes and an array ​countArray is initiated to the length of​ needles.

if (needles.length > 5) { System.err.println("Too many words!"); } else { int[] countArray = new int[needles.length];


Then, with a for loop, the function splits the string​ haystack using the delimiters whitespace, quotation marks, tab characters, newlines, form feeds, and carriage returns​. A new array​, words is created where the remaining elements are stored.

for (int i = 0; i < needles.length; i++) { String[] words = haystack.split("[ \"\'\t\n\b\f\r]", 0);


In a nested for loop, for each iteration, the elements in​ words are compared to the elements in​ needles. ​ If the two are equal, the value in​ countArray is incremented by 1.

for (int j = 0; j < words.length; j++) { if (words[j].compareTo(needles[i]) == 0) { countArray[i]++; }


At the end of the loop, the function outputs the matched elements and the corresponding value in​ countArray separated by​ “:”.

for (int j = 0; j < needles.length; j++) { System.out.println(needles[j] + ": " + countArray[j]);


You have now calculated the needle in the haystack!
